# Task ID: 3
# Title: Backend: Develop Core Statistical Analysis Engine
# Status: in-progress
# Dependencies: 2
# Priority: high
# Description: Build the engine to perform statistical comparisons between two identified test periods ('n' and 'n-1').
# Details:
As per PRD Section 3.2, for each peg/cell, calculate the key metrics: Average, Delta, Standard Deviation, Relative Standard Deviation (RSD), and Z-Score. Implement the data aggregation logic for summary peg groups. This module will take two dataframes (for n-1 and n periods) as input and output a dataframe with the calculated statistics.

# Test Strategy:
Create unit tests with pre-calculated expected outputs for a sample dataset. Validate the correctness of each statistical formula. Test edge cases like zero division for RSD calculation or periods with no data.
